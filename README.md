# TypoGen

Simple typo generator (English)

## Installation

```
$ gem install typogen
```

## Usage

```
$ typogen [word]
```

## Example

```
$ typogen install
```

output

```
nstall
istall
intall
insall
instll
instal
nistall
isntall
intsall
insatll
instlal
instale
```
